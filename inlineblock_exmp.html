The display Property

display: inline

Displays an element as an inline element. Any height and width properties will have no effect. Display inline Displays an element as an inline element. Any height and width properties will have no effect.

display: inline-block

Display an element or list items horizontally instead of vertically.Display inline-block gravida nisl sit amet facilisis. Nullam cursus fermentum velit sed laoreet.

display: block

Displays an element as a block element. It starts on a new line, and takes up the whole width. Display block Displays an element as a block element. It starts on a new line, and takes up the whole width.
